# 짝궁 (Jjak-gung) - 사주 데이팅 앱

![짝궁 로고](https://images.unsplash.com/photo-1614850715649-1d0106293bd1?q=80&w=1000&auto=format&fit=crop)

사주 명리학을 기반으로 운명적인 인연을 찾아주는 하이브리드 데이팅 플랫폼 **'짝궁'**입니다.

## 🔗 배포 주소
- **Vercel**: [https://jjakggung-oy85ccyny-jj-akgung.vercel.app/](https://jjakggung-oy85ccyny-jj-akgung.vercel.app/)

## ✨ 주요 기능
- **사주 기반 매칭**: 사용자의 생년월일시와 오행 분석을 통한 정밀 궁합 산출.
- **프리미엄 UI/UX**: 글래스모피즘과 밤하늘 애니메이션이 적용된 고품질 디자인.
- **상세 프로필**: 인연의 상세 사주 리포트와 연애 스타일 분석 제공.
- **SPA 아키텍처**: 홈, 매칭, 커뮤니티, 내 사주 등 원활한 메뉴 전환.

## 🛠 기술 스택
- **Frontend**: Vanilla HTML5, JavaScript (ES6+), Tailwind CSS (CDN).
- **Styling**: Google Fonts (Inter, Noto Sans KR), Custom CSS Animations.
- **Infrastructure**: Vercel Deployment, Git Version Control.
